## Arquivos principais

- `index.html`: estrutura, textos, SEO e CTAs
- `style.css`: estilos originais e componentes
- `fixes.css`: correções responsivas anteriores
- `brand.css`: identidade visual DUDU
- `script.js`: interações e animações
- `robots.txt`: orientação para buscadores
- `sitemap.xml`: URL principal para indexação
- `.nojekyll`: compatibilidade com GitHub Pages

## Publicação no GitHub Pages

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
